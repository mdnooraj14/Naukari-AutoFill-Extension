### Detailed vs Simple Mode
- **Toggle checkbox**: "☑ Detailed Prompt"
- **Checked** = Full format (Context, Task, Requirements, Output)
- **Unchecked** = Simple format (Task & Requirements only)

## 🐛 Troubleshooting

### Icon Doesn't Show in Toolbar
```
1. Chrome menu (top right) → Settings
2. Extensions (left sidebar)
3. Find "Prompt Enhancer"
4. Click PIN icon next to it
5. Now it appears in toolbar
```

### Extension Shows Error
```
1. Right-click on extension popup
2. Click "Inspect"
3. Check the Console for error messages
4. Common fixes:
   - Make sure all files are in the folder
   - Check file names are exact (case-sensitive)
   - Reload extension (disable/enable)
```

### Can't Find "Load unpacked" Button
```
1. Refresh chrome://extensions/ page
2. Make sure Developer mode is ON
3. Should appear in top left area
```

### History Not Working
```
1. Check if Storage permission is granted
2. Try clearing Chrome cache
3. Disable and re-enable extension
4. Open DevTools (F12) and check console
```
