# ⚡ Quick Setup Guide - Prompt Enhancer

## 🚀 5-Minute Installation

### Copy-Paste Instructions

```
1. Open Chrome Browser
2. Type in address bar: chrome://extensions/
3. Top-right corner: Toggle "Developer mode" ON
4. Click "Load unpacked" button
5. Navigate to: D:\Naukari-Bot\prompt-enhancer-extension\
6. Select the folder
7. Done! ✅
```

## 🎯 First Use

```
1. Click the extension icon in Chrome toolbar
2. Paste any prompt in the text box
3. Click "✨ Enhance Prompt"
4. See the improved version
5. Click "📋 Copy" to copy it
6. Paste into Claude, ChatGPT, or any AI tool
```

## 📌 Troubleshooting

| Problem | Solution |
|---------|----------|
| Can't find extension | Click puzzle icon → Find "Prompt Enhancer" → Pin it |
| Developer mode missing | Refresh chrome://extensions/ page |
| "Load unpacked" button missing | Make sure Developer mode is ON |
| All files show errors | Verify all 6 files are in the folder |
